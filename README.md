# Interior Catalog Project

This project is a comprehensive interior catalog for Unreal Engine 4 (UE4), containing a wide range of furniture items and their management through C++ and C#.

## Project Structure

- **Source/**: Contains C++ source files for functionality.
- **CSharp/**: Contains C# scripts for data management and models.
- **Tests/**: Contains test files to ensure project reliability.
- **Data/**: Contains sample JSON catalogs.
- **Config/**: Contains configuration files.

### Requirements
- Unreal Engine 4
- C++ Compiler
- .NET Framework

## Getting Started
1. Clone the repository.
2. Open the project in Unreal Engine 4.
3. Build the project to start using the catalog.
